# Logo Placement Instructions

## Kaduna State Government Logo

Place the official Kaduna State Government logo file in this directory (`public/`) with the name:

**`kaduna-logo.png`**

### Requirements:
- Format: PNG (preferred) or SVG
- Recommended size: At least 200x200px for crisp display
- Transparent background (if possible)

The logo will be used in:
- Login page header
- Error and loading states on auth screens
- Any other authentication-related UI components

### Update Path:
After placing the logo file, the image path used in components will be `/kaduna-logo.png` (Next.js automatically serves files from the `public/` directory).
